## PXI-2514 Custom Device ##

**PXI-2514 Custom Device** is a custom device for the NI fault insert unit PXI-2514 and PXIe-2514. 

### LabVIEW Version ###

LabVIEW 2020

### Built Availability ###

This IP is built and available for NI VeriStand 2020 and later.
For VeriStand 2020 or later: See Releases tab.

### Quality, Limitations ###

This IP should be considered mature, but of low quality. The code is validated on a Linux RT target with Linux RT System Image 2023 Q4. 

### Dependencies ###

NI-SWITCH
